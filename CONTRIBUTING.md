# Contributing

Thank you for your interest in contributing!

## Getting started

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Commit using [Conventional Commits](https://www.conventionalcommits.org/)
5. Open a Pull Request

## Development setup

```bash
git clone https://github.com/xabierlameiro/englishGame.git
cd englishGame
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Quality gates

```bash
npm run lint    # ESLint
npm run build   # Next.js production build
```

## Reporting bugs

Open a [GitHub issue](https://github.com/xabierlameiro/englishGame/issues) with clear reproduction steps.
