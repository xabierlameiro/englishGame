# English Game — Irregular Verbs Quiz

An interactive web game to practice English irregular verbs, built with Next.js 13 and TypeScript.

[![CI](https://github.com/xabierlameiro/englishGame/actions/workflows/ci.yml/badge.svg)](https://github.com/xabierlameiro/englishGame/actions/workflows/ci.yml)

## Stack

| Layer           | Choice              |
| --------------- | ------------------- |
| Framework       | Next.js 13          |
| Language        | TypeScript          |
| Styling         | CSS Modules         |
| Package manager | npm                 |
| Deployment      | Vercel              |

## Getting started

```bash
git clone https://github.com/xabierlameiro/englishGame.git
cd englishGame
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Script          | Description               |
| --------------- | ------------------------- |
| `npm run dev`   | Start development server  |
| `npm run build` | Production build          |
| `npm run start` | Start production server   |
| `npm run lint`  | ESLint                    |
